# Настройка команд бота в Telegram

## Как добавить команды в меню бота

1. **Найдите BotFather в Telegram**
   - Ищите `@BotFather` или перейдите по ссылке: https://t.me/botfather

2. **Выберите вашего бота**
   - Отправьте `/mybots` чтобы увидеть список ваших ботов
   - Выберите нужного бота

3. **Установите команды**
   - Отправьте `/setcommands`
   - BotFather попросит выбрать бота (если у вас их несколько)
   - Скопируйте и вставьте следующий текст:

```
help - Показать справку по всем командам
balance - Посмотреть свой баланс и сумму в банке
bank - Управление банковским счетом
inv - Посмотреть инвентарь плашек
list - Список активных участников игры
prize - Показать текущую плашку приза
leaderboard - Доска лидеров по стоимости инвентаря
reset - Сбросить текущий раунд игры
game - Начать автоматическую игру с таймером
stopgame - Остановить текущую игру
```

4. **Подтвердите**
   - BotFather подтвердит установку команд

## Результат

После установки команд, когда пользователи начнут писать сообщение в чате с ботом, появится меню с кнопками этих команд. Пользователи смогут просто нажать на нужную команду вместо того, чтобы печатать её вручную.

## Примечание

Команды с параметрами (например, `/coin 1 100`, `/pay @user 500`, `/add Имя Фамилия username`) оставлены для ручного ввода, так как они требуют дополнительных аргументов.
